# Release Notes

## Version 1.0.0 - Initial Release

### Features
- Introduced the `JsonMergePatch` library for applying JSON Merge Patch operations.
- Added a Roslyn source generator to create `UpdateDto` classes and `ApplyTo` methods at compile time.
- Supports `System.Text.Json` for JSON handling.
- Compatible with `DotNext.Optional<T>` for optional value handling.
- Targets `netstandard2.0` for broad compatibility.

### Testing
- Comprehensive test coverage using NUnit.
- Validated compatibility with `JsonSerializer` and `DotNext.OptionalConverterFactory`.
